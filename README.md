## Dependencies
```
sudo apt-get install libgtk-3-dev
```

## Install
```
git clone https://github.com/yuan-0816/mis-ec-UI.git
cd mis-ec-UI/build
cmake ..
make
```

### Usage
```
sudo ./msi_ec_control_gui
```
